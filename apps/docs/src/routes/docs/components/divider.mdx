# Divider

Separates two areas with a horizontal or vertical line and an optional label.

## Import

```tsx
import { Divider } from "@hope-ui/core";
```

## Usage

The `Divider` displays a thin horizontal or vertical line, and renders as an `<hr>` by default.

### Variants

Use the variant prop to change the visual style of the divider.

<Preview>
  <VStack spacing={4}>
    <Divider variant="solid" />
    <Divider variant="dashed" />
    <Divider variant="dotted" />
  </VStack>
</Preview>

```tsx
<VStack spacing={4}>
  <Divider variant="solid" />
  <Divider variant="dashed" />
  <Divider variant="dotted" />
</VStack>
```

### Thickness

Use the `thickness` prop to change the thickness of the divider.

<Preview>
  <Divider thickness="3px" />
</Preview>

```tsx
<Divider thickness="3px" />
```

### Vertical orientation

You can make a vertical `Divider` by passing `vertical` to the `orientation` prop.

<Preview>
  <HStack spacing={4}>
    <span>One</span>
    <Divider orientation="vertical" />
    <span>Two</span>
    <Divider orientation="vertical" />
    <span>Three</span>
  </HStack>
</Preview>

```tsx
<HStack spacing={4}>
  <span>One</span>
  <Divider orientation="vertical" />
  <span>Two</span>
  <Divider orientation="vertical" />
  <span>Three</span>
</HStack>
```

## Theming

Use the `DividerTheme` type to override `Divider` styles and default props when extending Hope UI theme.

```tsx
import { extendTheme, DividerTheme } from "@hope-ui/core";

const theme = extendTheme({
  components: {
    Divider: {
      // ...overrides
    } as DividerTheme,
  },
});
```

### Component parts

| Name         | Static selector             | Description           |
| ------------ | --------------------------- | --------------------- |
| root         | `hope-Divider-root`         | Root element          |
| labelWrapper | `hope-Divider-labelWrapper` | Wrapper arround label |

## Props

| Name           | Type                                | Description                           | Default value |
| -------------- | ----------------------------------- | ------------------------------------- | ------------- |
| variant        | `"solid" \| "dashed" \| "dotted"`   | The visual style of the divider.      | "solid"       |
| thickness      | `ResponsiveValue<number \| string>` | The thickness of the divider (in px). | 1px           |
| orientation    | `"horizontal" \| "vertical"`        | The orientation of the divider.       | "horizontal"  |
| labelPlacement | `"left" \| "right" \| "center"`     | The placement of the label, if any.   | "center"      |
