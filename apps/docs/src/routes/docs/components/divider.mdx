# Divider

Divider is a line that separates two areas.

## Import

```tsx
import { Divider } from "@hope-ui/core";
```

## Usage

<Preview>
  <Divider />
</Preview>

```tsx
<Divider />
```

### Divider with content

Using a `Divider` with content is normal operation.
You can pass `textPosition` prop to decide the position of label text.
In addition, with `plain` prop the label text will render as plain text instead of title.

<Preview>
  Pax mundi, seu pax in Terra, est notio optimae pacis intra et inter omnes homines et gentes in
  Planeta Terra.
  <Divider />
  Pax mundi, seu pax in Terra, est notio optimae pacis intra et inter omnes homines et gentes in
  Planeta Terra.
  <Divider>CENTER</Divider>
  Pax mundi, seu pax in Terra, est notio optimae pacis intra et inter omnes homines et gentes in
  Planeta Terra.
  <Divider textPosition="left">LEFT</Divider>
  Pax mundi, seu pax in Terra, est notio optimae pacis intra et inter omnes homines et gentes in
  Planeta Terra.
  <Divider textPosition="right">RIGHT</Divider>
  Pax mundi, seu pax in Terra, est notio optimae pacis intra et inter omnes homines et gentes in
  Planeta Terra.
  <Divider plain>plain</Divider>
  Pax mundi, seu pax in Terra, est notio optimae pacis intra et inter omnes homines et gentes in
  Planeta Terra.
</Preview>
```tsx
<Box>
  <Divider />
  {content}
  <Divider>CENTER</Divider>
  {content}
  <Divider textPosition="left">LEFT</Divider>
  {content}
  <Divider textPosition="right">RIGHT</Divider>
  {content}
  <Divider plain>plain</Divider>
  {content}
</Box>
```

You can make a vertical `Divider` using the `orientation` prop.

## Props

| Name         | Type                            | Description                                     | Default value |
| ------------ | ------------------------------- | ----------------------------------------------- | ------------- |
| dashed       | `boolean`                       | Whether the divider line should be dashed line. | false         |
| orientation  | `"horizontal" \| "vertical"`    | The orientation of divider.                     | horizontal    |
| plain        | `boolean`                       | Whether the lable text should be plain text.    | false         |
| textPosition | `"center" \| "left" \| "right"` | The position of label text                      | center        |
