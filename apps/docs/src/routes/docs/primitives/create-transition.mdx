# createTransition

Animate the presence of a component with predefined or custom transitions.

## Import

```tsx
import { createTransition } from "@hope-ui/core";
```

## Usage

The `createTransition` primitive allow you to work with enter/exit transitions. It comes with predefined transitions and option to create your own based on CSS properties.
